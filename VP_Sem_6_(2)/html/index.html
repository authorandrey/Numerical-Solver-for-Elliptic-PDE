Numerical Solver for -a u_xx - b u_yy = f

Overview

This project implements a numerical solver for the elliptic partial differential equation

\[-a \frac{\partial^2 u}{\partial x^2} - b \frac{\partial^2 u}{\partial y^2} = f(x,y) \]

on the unit square \([0,1]\times[0,1]\) with Dirichlet boundary conditions.

The solution is obtained using the Seidel method, and the eigenvalues of the discrete Laplacian operator are computed using the power method and shifted power method.

Main Modules

  • seidel_solver.h : Implementation of the Seidel iterative solver.
  • power_method.h : Power method for the largest eigenvalue.
  • shifted_power_method.h : Shifted power method for the smallest eigenvalue.
  • laplacian_operator.h : Discrete Laplacian operator.
  • functions.h : Exact solution, source term, and boundary conditions.
  • eigen_theory.h : Theoretical eigenvalues for verification.
  • output_handler.h : Output routines (saving to binary file, plotting).
  • solver_controller.h : High-level controller that runs the complete simulation.

Usage

The program is executed via main.cpp, which sets up the parameters:

  • Coefficients: \(a = 0.9\), \(b = 1.1\)
  • Grid step: \(h = 0.1\)
  • Tolerances: \(10^{-6}\) for both eigenvalue and Seidel iterations
  • Maximum iterations: 100000

The solver:

  1. Solves the PDE using the Seidel method.
  2. Saves the solution to solution.bin and generates plots via plot_full.py.
  3. Computes the largest eigenvalue (using alternating-sign initial vector) and the smallest eigenvalue (using unit constant initial vector).

Notes

  • The grid is uniform with \(N = 1/h + 1\) nodes per side.
  • Theoretical eigenvalues are given by:

    \[ \lambda_{\min} = \frac{4(a+b)}{h^2}\sin^2\frac{\pi h}{2},\qquad \lambda_{\max} = \frac{4(a+b)}{h^2}\cos^2\frac{\pi h}{2}. \]

  • The Python script plot_full.py requires numpy and matplotlib.
