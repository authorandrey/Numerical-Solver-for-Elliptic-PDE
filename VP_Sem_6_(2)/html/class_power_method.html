std::tuple< DataType, unsigned > PowerMethod::compute_max ( Params< Operator > & params) const
inline

Computes the largest eigenvalue using the power method.

Parameters
paramsParameters, including the initial vector and operator.
Returns
Tuple (eigenvalue, number of iterations).

The initial vector is normalized, then iterations: x_{k+1} = A x_k / ||A x_k||, lambda = x_k^T A x_k. Stopping criterion: residual ||A x - lambda x|| / |lambda| < epsilon.
