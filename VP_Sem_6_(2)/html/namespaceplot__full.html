◆ read_full_solution()

Tuple[np.ndarray, np.ndarray, np.ndarray] plot_full.read_full_solution ( str filename)

Reads a binary solution file created by OutputHandler.save_full_solution.

Parameters
filenameFile name (e.g., 'solution.bin').
Returns
Tuple (X, Y, u_num), where X, Y are coordinate matrices (meshgrid), u_num is a 2D array of the numerical solution.
Exceptions
ValueErrorif the file is too short or has an invalid format.

File format:

  • first 8 bytes: number of nodes along x (nx, size_t)
  • next 8 bytes: number of nodes along y (ny, size_t)
  • then nx*ny double values in row-major order (j=0..ny-1, i=0..nx-1)
