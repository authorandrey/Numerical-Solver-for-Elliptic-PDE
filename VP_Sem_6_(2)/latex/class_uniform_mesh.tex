\doxysubsection{Detailed Description}
Represents a uniform rectangular grid on \mbox{[}0,1\mbox{]}x\mbox{[}0,1\mbox{]} with a given step. 

The grid is square, nodes at points (i\texorpdfstring{$\ast$}{*}h, j\texorpdfstring{$\ast$}{*}h) where i,j = 0,...,N-\/1, h – actual step, N = 1/h + 1.
